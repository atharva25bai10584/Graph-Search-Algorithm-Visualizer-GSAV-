## Overview of the Project

The Graph Search Algorithm Visualizer is an interactive Python-based project that helps users understand how different search algorithms work on graphs.

It allows users to select start and goal nodes, choose an algorithm, and visualize the step-by-step traversal along with logs and final results.

---

## Features

Visualizes BFS, DFS, UCS, and A* algorithms

Interactive selection of start and goal nodes

Step-by-step graph visualization

Highlights visited nodes, frontier, and final path

Displays traversal logs

Supports weighted graphs

Heuristic-based A* search

Run the first cell in the notebook:

# Step 2: Run the notebook

# Step 3: Use the application
